Setting Up Your Server

  1. Download the latest Minecraft server JAR
  2. Create a folder and place the JAR inside
  3. Run the JAR once to generate files, then accept the EULA
  4. Configure `server.properties` to your liking
  5. Port forward (if hosting locally) or use a hosting service
  6. Start your server and invite friends!
